# Multi Timer App

Простое десктоп-приложение для учёта рабочего времени и формирования отчётов.

Подходит для:
- ведения задач
- подготовки еженедельных отчётов
- трекинга времени

## ✨ Возможности

- ⏱ Создание нескольких таймеров
- ▶️ Запуск / пауза каждого таймера отдельно
- ✏️ Редактирование названия задачи
- 🗑 Удаление таймеров

- 🔄 Drag & Drop (перетаскивание таймеров)
- 💾 Автоматическое сохранение состояния

- 📊 Разделение задач:
  - Плановые
  - Дополнительные

- 📤 Экспорт в Excel:
  - Разделение по типам задач
  - Время по каждой задаче

- ⌨️ Горячие клавиши:
  - Ctrl + Z — восстановление удалённого таймера

- 🎨 Удобный минималистичный интерфейс
