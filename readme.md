⏱️ Multi Timer

Простое и удобное desktop-приложение для учета времени по задачам.
Разработано на Electron с упором на скорость, удобство и безопасность.

🚀 Возможности
Таймеры
До 12 таймеров в одном окне
Одновременно активны не более 2
Быстрый старт / пауза
Индивидуальные названия задач

Типы задач
🟥 Плановые
⬜ Дополнительные

Используются в статистике и экспорте.
