# ⏱ Multi Timer

Простое и удобное десктопное приложение для учета времени задач.

Создано для работы с несколькими таймерами одновременно, с возможностью экспорта в Excel и ведения архива.

---

## 🚀 Возможности

- До 12 таймеров одновременно
- Максимум 2 активных таймера
- Быстрый запуск через шаблоны
- Автоматическое добавление в архив при удалении
- Экспорт в Excel (разделение: плановые / дополнительные)
- Архив с возможностью выгрузки
- Темная / светлая тема
- Работа в системном трее
- Frameless UI (кастомное окно)

---

## ⚙️ Логика работы

### Таймеры

- Таймер можно запустить только с названием
- При запуске 3-го таймера — один из активных автоматически ставится на паузу
- Название нельзя менять во время работы

### Типы задач

- Плановая
- Дополнительная

Используются для разделения в экспорте

---

## 📦 Шаблоны

- Быстрое создание таймера
- Ограничение: до 7 шаблонов
- Запуск таймера сразу при выборе
